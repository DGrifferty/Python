A class based solution to 1D Linear Advection equation implementing FCTS, Lax and Lax Wendroff methods

see code to reproduce animation figures.py for examples on how to use the program

set_plot_directory, set_animation_directory, set_results_directory, set_record_to_file can be called
to change the respective directory for recording files, animations and plots

set_c, set_x_axis_width, set_pulsewidth, set_dx, set_dt can be called to change variables

plot will plot the current grid, if name is not provided the name of the last method called will
be set as the plot name. If plot expected is set to false, 

create_animation
